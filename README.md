# wgpu4k Hello Cube Sample

Minimal project to show how to use wgpu4k 

## How to Run the Project

1. On desktop: `./gradlew runJvm`
2. On web js: `./gradlew jsRun`
2. On web wasm: `./gradlew wasmJsRun`
